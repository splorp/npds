Appendix: CGI Environment and Request Variables

Variable

String Returned

SERVER_SOFTWARE

nHTTPd/2.02 (Newton)

SERVER_NAME

IP Address of Server

SERVER_PROTOCOL

HTTP/1.0

GATEWAY_INTERFACE

CGI/1.0 (NPDS_SSA)

SERVER_PORT

Current Server Port

REMOTE_ADDR

Client IP Address

REMOTE_HOST

Client DNS (If Resolved*)

HTTP_USER_AGENT

Client Browser

SCRIPT_NAME

Title of Current Script

*NPDS does DNS lookup in an odd way to increase its speed. It looks up the first instance of an IP AFTER that address makes its first request. Therefore, you can't have a script that uses the DNS entry for a user located on the first page that user will access. Also note that if the DNS lookup failed, this service will not be available and will return instead the IP address of the user.
