Binary Server 1.11

This server allows you to serve binaries such as images, compressed archives, and documents from a soup.

BinInstaller 1.2

Binaries are installed in the soup using BinInstaller, an Newton Toolkit project script. You compile BinInstaller for each binary you want to have available and then install each compiled package onto the Newton.

Preliminary documentation of these modules is covered in GIF Server Syntax.

Datebook Server 2.12

View Datebook Server Software

GIF Server 1.3

This is a 16-level grayscale (or greenscale) GIF Server, which lets you serve a screen shot of your Newton. When the backlight is turned on, the screen shot is black on green. It’s much faster than the GIF Server released with the original version 2.0 NPDS modules.

Version 1.3 of the GIFServer module provides support for serving built-in bitmaps directly from the NewtonOS ROM. Preliminary documentation of this feature is covered in GIF Server Syntax.

View GIF Server Software

Notepad Server 2.104

This server allows you to serve data created the note, outline or checklist stationary from the Notes application. General features and configuration for the 2.0 version of the Notepad Server module are covered in the NPDS 2.0 Documentation.

Updated documentation regarding the set up and configuration of the Notepad Server module is covered in Notepad Server Configuration.

NPDS Watcher 1.1a1

This module watches the server and reboots it when the server crashes or stops responding to requests.

NPDS 2.1 and later requires latest NPDS Watcher 1.1a1.

Utilities

Debug 500 Page 1.0

This module patches the default NPDS 500 error page by adding the name of the exception that occurred. A status code 500 is returned when the server encounters an unexpected condition preventing it from fulfilling the requested action. More information regarding HTTP Status Codes can be found on the W3C site.

Script Editor 2.1

The Script Editor allows you to create NewtonScript scripts for generating dynamic content on your Newton. Several other user-submitted scripts are provided as examples, in addition to the ones included in the NPDS Scripts package. A good introductory scripting tutorial is available in the 4AA Magazine.

ProtoSoupBrowser 1.1

This is the layout code for the cache browser used in the nHTTPd module.

Wipe 1.02

The Wipe utility has been updated to remove preferences of new modules. It also removes a ghost preference left by the NPDS server core.

A copy of the BugTrap debugging utility from Tactile Systems has also been included in this section. BugTrap can be used to help determine and isolate crashes and errors produced by the NPDS software. Please note that this package is not supported by the NPDS development team. Documentation is supplied with the BugTrap package.

AutoCloseNotif

The AutoCloseNotif package will automatically close most notification windows and card slips after a soft reset. Useful for NPDS users that have NPDS Watcher set to reboot their Newton, and would like the backdrop visible for those nifty screen shots. Additional background information is available on the Newton Dose site. Written by Daniel Padilla Roldán

View Utilities Software

Web Pager 2.1

This module allows you to be paged by visitor through your Newton. Upon receiving a page, the Newton backlight flashes and a message is displayed. The message will also be spoken through the Newton’s built-in speaker if you have the MacInTalk and SpeakText packages installed. Features and configuration for the 2.0 version of the Web Pager module are covered in the NPDS 2.0 Documentation.

View Web Pager Software
